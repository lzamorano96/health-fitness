## Estimation Rules

When an item isn't listed above:
1. Use the closest analog in this table as a baseline
2. Set `"estimated": true` in the log entry
3. Note the assumption in the `notes` field (e.g., `"assumed medium portion, similar to chicken breast"`)
4. Never fabricate precise values — round to nearest 5 kcal, nearest gram for macros
