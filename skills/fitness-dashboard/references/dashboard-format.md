# Dashboard Export Format

When the user asks to export the dashboard, write a markdown file to:
`data/reports/YYYY-WNN-report.md`

## Export Template


## Status Labels

| Condition | Label |
|-----------|-------|
| Within ±5% of calorie target | `✓ On target` |
| 5–15% below target | `↓ Low` |
| 5–15% above target | `↑ High` |
| >15% below target | `↓↓ Significantly low` |
| >15% above target | `↑↑ Significantly high` |
| No log file | `No data` |
